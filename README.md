TEG-COMSOL-Simulation-Based-Research
This repository is a curated collection of resources focused on the modeling and simulation of Thermoelectric Generators (TEGs) using COMSOL Multiphysics. It includes original educational material and a selection of academic papers to support research in energy harvesting and waste heat recovery.
